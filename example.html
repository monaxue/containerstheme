Panels tabset

Panel tabset

  • Style
  • Automatic advance
  • These are stylized panel tabsets
  • Defined by wrapping with ::: {.panel-tabset}
  • Usel level-2 heading (## Tab Name) to define each tab
  1. Tabs are automatically advanced
  2. This feature comes from a bundled plugin
    • RevealJS Tabset plugin

Columns

Columns

  • These are stylized columns with boxes around each column
  • You can set the box in the column to be nonincremental using
    • ::: {.column .nonincremental}
  • By setting as nonincremental
    1. The boxes can be seen first, before any of the interior the items
    2. Otherwise, the boxes advance like a normal fragment

Color

Background color

  • Set title slide color in the yaml using 
    • title-slide-attributes:
    data-background-color: "your-color-of-choice"
  • A new background color is set using
    • ## Section title {background-color="color-of-choice"}
    • Otherwise, it inherits the background color of the previous slide

Secondary color

  • Secondary colors for containers in lists, panels, and columns, are automatically calculated and set from the background color
